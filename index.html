Intro

Hey, I'm CalledCracki, a passionate app and game developer. I'm glad you found your way to my portfolio. I've been spending a lot of time programming for a few years now, including websites, games and extensions for the sandbox game Minecraft using the Java programming language.

Since I attach great importance to anonymity on the internet, I began to steadily expand my programming skills early on under the pseudonym "CalledCracki". Thus, I can now boast a fairly impressive repertoire of Github repositories, which I regularly expand. In order to make my skills available to others, I recently started offering services on Fiverr. In the future, I hope to convince people of my knowledge professionally as well.

If you want to get a personal impression of my work, don't hesitate to have a look here. If you have any questions, please feel free to contact me at any time.

Projects


Wardrobe Plus

Wardrobe Plus is a system inspired by the largest Minecraft server, which allows a player to store armour in a GUI and put it on there automatically.


GGFeatures Reloaded

GGFeatures Reloaded is based on the original system by Niklas409. It contains all the important features of probably the biggest German MC network.




Cuboid

Cuboid was the first Unity game I actively worked on, mainly to familiarise myself with Unity as a game engine and to get started programming with C#.


Aimazing

Aimazing is the latest Unity game I'm working on. I originally started programming this game for fun at the request of a friend, but I liked the idea of my own Aimtrainer so much that I decided to work on it more often.
It also gives me the opportunity to explore universal render pipelines and post processing features.

End of Time Website

I "programmed" the "End of Time" website to illustrate how long it takes for the time system of some computer systems to run out. Since some systems use Unix time, it is possible to say approximately when the so-called "overflow" error will occur.
